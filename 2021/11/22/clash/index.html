配置

1.注册

登录 — GW树洞 (hello-shudong.com) ：邀请码gayP

2.依据官方教程安装

3.变更套餐

clash 新用户会赠送1G流量，可以在Profiles内看到

  • 点击【钱包】充钱，之后点击【商店】购买套餐

  • 之后在Profiles内刷新一下，核对流量是否和套餐一致

此处界面可查看当前流量信息；

当套餐变更时 需要刷新一下。

  • 如果刷新不行，还可以在主页内重新导入订阅地址，进行设置

4.注意

  • 选择Global ，所有流量都会走代理（只能访问外网，不能访问国内的苏大网），使用Rule可内可外

  • GitHub 需要特别设置一下

在命令行界面内使用git clone 或pull 等获取远程数据的 命令，需要设置代理加速；此处使用clash for window 设置代理，其端口为7890
